ShopKart is a prototype e-commerce web application developed using Django and MySQL. The project demonstrates core e-commerce functionalities such as product listing, shopping cart management, and user interaction with a responsive frontend interface.

 Features

Product listing with dynamic data

Add-to-Cart functionality

Responsive UI using Bootstrap 5

User-friendly navigation

Database integration using MySQL

Admin panel for managing products

 Tech Stack

Backend: Python, Django

Frontend: HTML, CSS, Bootstrap 5

Database: MySQL
